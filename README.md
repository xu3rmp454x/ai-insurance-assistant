# 🛡️ AI 車險理賠輔助工具

一個結合 AI 的車險理賠分析工具，輸入事故描述，即時獲得專業的理賠建議與文件清單。

## 功能特色

- **即時 AI 分析**：輸入事故描述，AI 串流即時輸出分析結果
- **專業理賠建議**：涵蓋台灣強制險、甲/乙/丙式車體險、第三人責任險
- **文件清單**：自動列出辦理理賠所需文件
- **責任判斷**：初步分析雙方肇事責任比例
- **流程說明**：提供理賠申辦步驟與預估時程

## 技術架構

| 層級 | 技術 |
|------|------|
| 前端 | HTML / CSS / JavaScript |
| 後端 | Node.js + Express |
| AI | Google Gemini API（串流輸出） |

## 專案動機

本人從事車險理賠工作多年，深知許多民眾在事故後不知道該如何處理、要準備哪些文件。這個工具結合 AI 與保險專業知識，希望能幫助民眾快速了解理賠方向。

## 本地執行

```bash
# 安裝套件
npm install

# 建立 .env 檔案
cp .env.example .env
# 填入你的 Gemini API Key

# 啟動伺服器
node server.js
```

開啟瀏覽器：`http://localhost:3000`

## 環境變數

```
GEMINI_API_KEY=你的 Google Gemini API Key
PORT=3000
```

---

> ⚠️ 本工具提供的分析僅供參考，不構成法律或保險專業建議，實際理賠結果依保單條款及現場勘查為準。
